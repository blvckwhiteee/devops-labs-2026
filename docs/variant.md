### Опис застосунку

**Notes Service** — це простий сервіс для зберігання текстових нотаток.

### Об’єкт нотатки містить наступні поля

* `id` - унікальний ідентифікатор;
* `title` - заголовок;
* `content` - основний текст;
* `created_at` - мітка часу створення.

### API сервісу складається з 3 ендпоінтів:
| Метод | Ендпоінт | Опис                                                 |
| :--- | :--- |:-----------------------------------------------------|
| `GET` | `/notes` | вивести список усіх нотаток (тільки `id` та `title`) |
| `POST` | `/notes` | Створити нову нотатку (`title`, `content`)           |
| `GET` | `/notes/<id>` | Вивести повний вміст нотатки                         |

### Інші вимоги:

- Назва застосунку незалежно від варіанту mywebapp

- Доступ до бази даних має бути обмежений лише з віртуальної машини

- Клієнти взаємодіють із системою через reverse proxy (nginx)
